# CLAUDE.md

Guidance for AI coding agents (e.g. Claude Code) working in this repository.
For the human-facing docs, see [CONTRIBUTING.md](CONTRIBUTING.md) and
[HACKING.md](HACKING.md).

## Pull request reviews

- After pushing changes that address a reviewer's feedback on a pull request,
  re-request a review from that reviewer (e.g. via the reviewer's re-request
  button, or the GitHub API). GitHub does not re-request automatically after a
  reviewer has left comments or requested changes, so without this the reviewer
  has to notice the update on their own and the PR can stall.
